## Key Features Implemented

### 1. Driver Management
- Registration with document upload
- Email/phone authentication
- Status tracking (Pending, Approved, Suspended)
- Real-time online status
- Rating system
- Commission rate flexibility

### 2. Order & Delivery System
- Order creation with items
- Status tracking (Pending → Delivered)
- Bolt-like commission model:
  - Base fare + Platform commission split
  - Driver gets 75% of base fare
  - Platform keeps 25%
  - Tips & bonuses go directly to driver
  - Peak hour multipliers

### 3. Real-time Tracking
- WebSocket-based live location
- GPS accuracy tracking
- Broadcast to multiple monitors
- HTTP fallback for location updates
- Last known location storage

### 4. Route Optimization
- FastAPI microservice architecture
- Distance calculation (Haversine formula)
- Duration estimation
- Alternative route generation
- Route caching (1-hour TTL, 5000 entry limit)
- Extensible for OSM/Google Maps integration

### 5. Admin Features

### 6. Security

## How to Run

### Quick Start (5 minutes)

1. **Setup virtualenv:**
```bash
python -m venv venv
.\venv\Scripts\Activate
pip install -r requirements.txt
```

2. **Initialize database:**
```bash
python -c "from app import create_app, db; app=create_app(); app.app_context().push(); db.create_all()"
```

3. **Run services (3 terminals):**
   - Terminal 1: `python app.py` (Port 5000)
   - Terminal 2: `python routing_service.py` (Port 8010)
   - Terminal 3: `go run live_location_service.go` (Port 5001)

4. **Access:**
   - http://localhost:5000 - Main app
   - http://localhost:5000/driver/login - Driver portal
   - http://localhost:5000/customer/login - Customer portal
   - http://localhost:5000/admin/dashboard - Admin dashboard
